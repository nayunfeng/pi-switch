# Pi Switch UI Prototype

This is a throwaway static prototype for validating Pi Switch UI flow before building the Tauri app.

Run:

```text
Open index.html in a browser.
```

Scope:

- No persistence.
- No Pi config writes.
- No real `pi -p "ping"`.
- No real `/models` request.
- State lives in memory.

Keep or delete this prototype after the UI decisions are absorbed into the real app.
